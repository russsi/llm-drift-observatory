Same name. Same model? The receipts, daily.

Every day, the same frozen battery of 36 tasks is sent to the same free-tier LLM APIs and graded by deterministic code — no LLM judges. If a provider silently changes the model behind a stable name, the line moves. Nobody can recreate this history later.

The method

An instrument, not a leaderboard.

The battery is frozen: tasks are never edited or removed once live. One day of scores means nothing — months of a flat line mean that when the line moves, something changed on the provider's side, and this page has the timestamped receipts.

  • Frozen public battery
  • Deterministic grading — no LLM judges
  • Scores never recomputed
  • Partial days excluded, never scored
  • False alarms stay on the record
  • Raw transcripts public per day
